## Installation

Install my-project with pip3

```bash
  cd OTP_CODE_API-main
  sudo pip3 install -r requirements.txt
```
    
## Usage/Examples

```python
python3 otp_main.py
```
This script reload every messages of the selected number every 15 seconds.
